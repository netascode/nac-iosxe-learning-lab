What you'll build

Four configuration scopes, one data model, one Terraform workflow.

Task 03 · Global Banner on every device Declare a setting once in global.nac.yaml. NaC applies it to all four switches at merge time. → iosxe.global.configuration Task 04 · Group ACL on the access switches Device groups target a subset of devices by role. The ACL deploys to access01 + access02 only. → iosxe.device_groups Task 05 · Device Loopback0 on core only Device-specific settings (router IDs, per-device IPs) live next to their device. Other devices are untouched. → iosxe.devices[name=core] Task 13 · Pipeline Validate, plan, deploy via GitLab A commit to main becomes a deploy. Merge requests preview without touching devices. GitOps for your network. → .gitlab-ci.yml
